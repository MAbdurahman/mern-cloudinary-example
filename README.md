# mern-cloudinary-example

## Features
- MongoDB, Express.js, ReactJS, and Node.js
- Authentication and Authorization with JWT
- Cloudinary


### Create config.env file in config directory
```env
PORT=
NODE_ENV=
FRONTEND_URL=
API_ENV=
MONGO_DB_URI=

JWT_SECRET=
JWT_EXPIRES_TIME=
JWT_LIFETIME=

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
CLOUDINARY_URL=cloudinary:

```
### Start the app
```shell
npm install
```
or 
```shell
yarn install
```
then
```shell
npm start
```
or 

```shell
yarn start
```

### Build the app
```shell
npm run build
```
or 

```shell
yarn build
```
